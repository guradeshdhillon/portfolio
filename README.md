Personal portfolio website for **Guradesh Dhillon** — a third-year Computer Engineering student at SVPCET specialising in AI & Data Science. Built as a zero-dependency, single-file site with a clean black-and-white editorial aesthetic, and deployed via GitHub Pages on a custom domain.

---

## Sections

- **Hero** — Full-bleed grayscale photo with animated identity badge tags
- **About** — Bio, academic background, and engineering philosophy
- **Projects** — Horizontally scrolling carousel of featured ML/AI and full-stack projects
- **Skills** — Technologies grouped by domain: ML/AI, Programming, Tools & Platforms, Web & Mobile
- **Experience** — Roles at ACM SVPCET, Unnat Bharat Abhiyan, and Engineering India
- **Certifications** — Industry simulations (BCG X, British Airways, TATA, DATACOM) and Kaggle credentials
- **Contact** — Email, phone, and social links

---

## Tech Stack

| Layer | Detail |
|---|---|
| Markup | HTML5 — single `index.html` |
| Styling | Vanilla CSS — custom properties, grid, flexbox, `backdrop-filter` |
| Fonts | Space Grotesk · Inter (Google Fonts) |
| Interactivity | Vanilla JavaScript — project carousel, scroll behaviour |
| Hosting | GitHub Pages with custom `is-a.dev` domain (via `CNAME`) |

No frameworks. No build step. Ships as a single file.

## Running Locally

No install or build step needed.

```bash
git clone https://github.com/guradeshdhillon/portfolio.git
cd portfolio
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```
